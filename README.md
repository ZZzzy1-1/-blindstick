## 📖 项目简介

**导盲杖随行助手**是一个面向视障人士的智能辅助出行系统。系统通过整合多种传感器数据（视觉、雷达、GPS）和AI语音交互技术，为用户提供实时的环境感知、导航指引和安全告警服务。

项目采用前后端分离架构，包含嵌入式硬件接口、Python后端服务和可视化大屏展示三个核心部分。

## ✨ 核心功能

### 1. 🎯 智能视觉识别（14类目标检测）
- **道路环境**：盲道、斑马线、马路牙子、楼梯台阶、水坑
- **交通信号**：红灯、黄灯、绿灯、标志牌
- **障碍物**：行人、车辆、立柱、垃圾桶、反光锥

### 2. 📡 多维度环境感知
- **雷达监测**：前方/左侧/右侧三向距离实时监测（<180cm自动告警）
- **GPS定位**：实时位置追踪、轨迹记录、卫星状态显示
- **障碍物告警**：语音播报障碍物方位和距离，智能建议避让方向

### 3. 🗺️ 智能导航系统
- **语音导航**：支持"带我去XX"语音指令启动导航
- **步行路线规划**：基于百度地图API的步行路径规划
- **实时步进导航**：根据GPS位置自动推进导航步骤
- **路线偏离检测**：自动检测偏离并重新规划路线

### 4. 🎙️ AI语音交互
- **语音识别**：百度ASR实时语音转文字
- **智能对话**：通义千问大模型提供自然语言交互
- **语音合成**：百度TTS语音播报导航和告警信息
- **唤醒词识别**：支持"带我去/我要去/导航到"等导航触发词

### 5. 📊 可视化数据大屏
- **实时视频流**：显示视觉检测结果和边界框
- **地图轨迹**：实时显示GPS位置和移动轨迹
- **设备状态**：主控/视觉/雷达/GPS/语音模块状态监控
- **今日出行统计**：总里程、导航次数、障碍物提醒、路线调整

## 🚀 快速开始

### 环境要求

- Python 3.8+
- pip 包管理器
- 现代浏览器（Chrome/Firefox/Edge）

### 1. 安装依赖

```bash
cd server
pip install -r requirements.txt
```

**主要依赖**：
- `flask` + `flask-sock` - Web服务器和WebSocket支持
- `requests` - HTTP请求库
- `urllib3` - 网络请求

### 2. 配置API密钥

编辑 `server/server.py`，填写你的API密钥：

```python
# 百度AI (语音识别 + 语音合成)
BAIDU_APPID = "你的APPID"
BAIDU_API_KEY = "你的API_KEY"
BAIDU_SECRET_KEY = "你的SECRET_KEY"

# 通义千问 (LLM对话)
DASHSCOPE_API_KEY = "你的DASHSCOPE_API_KEY"

# 百度地图 (导航服务)
BAIDU_MAP_AK = "你的百度地图AK"
```

### 3. 启动服务器

```bash
cd server
python server.py
```

服务器启动后会显示：
```
============================================================
 🚀 AI智能导盲杖中枢服务器 启动成功！
============================================================
 🔗 [本地大屏链接] 点击访问:  http://localhost:8080
 🔗 [局域网大屏链接] 点击访问: http://192.168.x.x:8080
```

### 4. 访问可视化大屏

在浏览器中打开显示的地址即可查看实时数据大屏。

## 📡 API接口说明

### 设备数据上报
```http
POST /api/data
Content-Type: application/json

{
  "radar": {
    "front": 120.5,   # 前方距离(cm)
    "left": 400.0,    # 左侧距离(cm)
    "right": 350.0    # 右侧距离(cm)
  },
  "gps": {
    "lat": 30.23193,  # 纬度
    "lng": 115.05791, # 经度
    "speed": 1.2,     # 速度(m/s)
    "satellites": 8   # 卫星数
  },
  "detections": [     # 视觉检测结果
    {"class": "person", "label": "行人", "x": 100, "y": 200, "w": 50, "h": 80, "confidence": 0.92}
  ]
}
```

### 语音交互
```http
POST /api/chat
Content-Type: audio/pcm

[PCM音频数据]

# 返回: WAV音频文件
```

## 🔧 配置说明

### 常住地设置

用户可以通过大屏设置常住地城市，导航搜索时会优先在该城市范围内查找目的地：

1. 点击右上角 ⚙️ 设置按钮
2. 输入常住地城市名称（如：黄石市）
3. 点击保存

配置文件保存在 `server/user_config.json`。

### 障碍物告警阈值

默认配置（可在 `server.py` 中修改）：
- **告警距离**：< 180cm 触发语音告警
- **冷却时间**：3秒内不重复播报相同告警
- **避让建议**：根据左右雷达距离智能推荐避让方向

### 导航路线偏离检测

- **偏离阈值**：30米
- **检查间隔**：每10秒检查一次
- **冷却时间**：30秒内不重复触发路线调整

## 🎯 视觉检测类别

系统支持检测以下14类目标：

| 类别 | 英文名 | 用途说明 |
|------|--------|----------|
| 盲道 | blind_track | 检测盲道位置，引导用户沿盲道行走 |
| 马路牙子 | curb | 检测道路边缘，防止跌落 |
| 斑马线 | crosswalk | 检测人行横道，安全过马路 |
| 立柱 | pole | 检测路灯杆、电线杆等障碍物 |
| 垃圾桶 | ashcan | 检测垃圾桶位置 |
| 反光锥 | reflective_cone | 检测施工警示锥 |
| 红灯 | red_light | 检测交通红灯 |
| 黄灯 | yellow_light | 检测交通黄灯 |
| 绿灯 | green_light | 检测交通绿灯 |
| 标志牌 | stop_sign | 检测交通标志 |
| 行人 | person | 检测前方行人，注意避让 |
| 车辆 | vehicle | 检测车辆，注意安全 |
| 楼梯台阶 | stairs | 检测楼梯，防止踩空 |
| 水坑 | puddle | 检测积水，防止滑倒 |
